# v2026.512.1

> Released: 2026-05-12

## Highlights

- **LLM Wiki plugin package** - the monorepo's `plugin-llm-wiki` package, fixtures, and tests stay wired into the build and release flow.
- **Sandbox provider packages** - Cloudflare and exe.dev sandbox provider packages now ship as explicit plugin packages with bridge templates and release-enrollment guardrails.

## Fixes

- **Issue comment attribution** - agent-posted comments with user author IDs now resolve to the correct agent author in issue threads, and system notice headers fall back to the CN brand when the run agent is unknown. ([#91](https://github.com/penclipai/paperclip-cn/pull/91))
- **Agent Pixels static assets** - plugin-key asset requests no longer crash the static UI route; valid assets resolve and missing assets return 404 instead of 500.

## Improvements

- **Release enrollment hygiene** - Cloudflare and exe.dev stay out of CI publishing until their one-time npm bootstrap is complete, keeping the fork's release flow predictable.
- **Fork sync polish** - the upstream sync branch keeps the current CN release surface aligned with the master merge.
