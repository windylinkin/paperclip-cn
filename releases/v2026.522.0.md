# v2026.522.0

> Released: 2026-05-22

## Highlights

- **`penclip` now starts from a dedicated CLI entrypoint** - the published binary no longer loads the plugin scaffolder as library code, which fixes startup failures during `penclip onboard --yes` and other CLI commands.

## Improvements

- **Plugin scaffolding stays compatible with installed npm layouts** - the scaffolder now resolves published SDK/shared packages when it is running from an installed CLI package, and the release smoke job checks the published bin entrypoint directly.

## Fixes

- **Plugin init no longer self-executes when imported** - the scaffolder library no longer has a top-level direct-run guard, so the published CLI wrapper owns command execution instead of the bundled library file.
- **Local plugin authoring docs point at the new CLI entrypoint** - `create-paperclip-plugin` usage examples now reference `dist/cli.js` instead of `dist/index.js`.
